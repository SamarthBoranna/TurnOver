# System Overview

## High-Level Architecture

* **Frontend**: Next.js (React) + Tailwind CSS (V0-generated)
* **Backend**: FastAPI (Python)
* **Database**: Supabase (Postgres)
* **Auth**: Supabase Auth

## Responsibilities

### Frontend

* UI rendering
* Client-side state
* API consumption

### Backend

* Recommendation logic
* Business rules
* Authorization enforcement

### Supabase

* Persistent storage
* Auth + RLS
